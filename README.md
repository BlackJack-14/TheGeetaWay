## Quick Start

```bash
# Clone repo
git clone https://github.com/YOUR_USERNAME/TheGeetaWay.git
cd TheGeetaWay

# Install
pip install -r requirements.txt

# Configure
python generate_api_key.py --update-env
# Add GROQ_API_KEY to .env

# Run
uvicorn api:app --reload
streamlit run app.py
```
